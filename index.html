Quick Step-By-Step

How To Generate Your Own Quotes And Illustrations

  • Click On Asset Care Or Annuity Care
  • Fill In Your Name In Presented By
  • Click On Asset Care
  • Complete The Entire Product Detail Portion
  • Click Calculate
  • The Quote Will Then Populate Under Result To The Right Side Of The Screen
  • To View The Illustration, Click Create PDF

Quick Step-By-Step

To Submit Your Own Application, Follow These Easy Steps

  • If You Have A User ID Through OneAmerica, Click Submit Application (With User ID)
  • If You Do Not Have A User ID Through OneAmerica, Click Submit Application (No User ID Available) And Follow The Instructions Needed To Start Writing An Application
  • If You Want To Run A Real Application, Click Run Sales Connection
  • If You Would Like To Practice Running A Sales Connection, Click Practice Sales Connection
  • To Start A New Application, Click Start New Case
  • To View Your Current Cases, Click View My Cases
  • Fill Out All Info As Requested Through The System
